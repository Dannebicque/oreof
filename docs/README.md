# Dossier docs - index

Ce dossier contient la documentation interne du projet ORéOF. Objectif : centraliser les guidelines UI, procédures d'exploitation et notes d'architecture.

Structure principale

- `ui-conventions/` : conventions UI, migration Tailwind et table de correspondance des icônes.
- `architecture/` : notes d'architecture, BDD et maquettes.
- `ops/` : procédures d'installation et commandes d'operation.
- `archives/` : documents historiques ou notes de travaux (à archiver ou nettoyer régulièrement).
